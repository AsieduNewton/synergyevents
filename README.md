Synergy Events 📅

Uniting Visions, Creating Moments.

Synergy Events is a premium event management and social engagement platform built to streamline event discovery, professional service booking, and digital memory sharing. This project showcases a high-fidelity, responsive frontend architecture designed with a "Mobile-First" philosophy.

🚀 Key Features

Interactive Landing Experience: A dynamic homepage featuring fade-up scroll animations and a glassmorphism navigation system.

Comprehensive Service Catalog: Detailed breakdown of event offerings including Event Planning, Ticket Sales, and Seat Reservations.

Memory Portal: A social-media-inspired gallery for post-event engagement and brand storytelling.

Virtual Ticketing Logic: A frontend implementation of a ticket selection and price calculation system.

Responsive Architecture: Fully fluid layouts optimized for Accra's diverse mobile and desktop user base.

🛠️ Technical Stack

Frontend: HTML5, CSS3 (Custom Properties & Modern Layouts)

Typography: Google Fonts Integration (Montserrat, Bebas Neue, Cormorant Garamond)

Interactions: Vanilla JavaScript (Intersection Observer API for scroll-based triggers)

Design System: A cohesive brand palette featuring high-contrast "Synergy Orange" and "Deep Purple" gradients.

🏗️ Project Architecture

├── index.html          # Core Discovery & Brand Introduction
├── about.html          # Company History, Milestones & Team Values
├── services.html       # Service Offerings & Price Selection Logic
├── memories.html       # Social Proof Gallery & Lightbox Interactions
└── images/             # Optimized Brand Assets (Logos & UI Previews)


🧠 Implementation Highlights

1. Performance-First Animations

Instead of heavy libraries, this project utilizes the Intersection Observer API to trigger "Fade-Up" animations only when elements enter the viewport. This ensures smooth performance on lower-end mobile devices common in emerging markets.

2. Modern UI/UX Patterns

The CSS architecture uses a robust Variable System (:root), allowing for instant global theme updates and consistent spacing across all four modules.
